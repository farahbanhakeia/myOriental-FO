Localiser sur la carte :

  • Une fois que vous avez ajouté les waypoints
  • Vous deviez saisir les laltitudes et longitudes puis
  • Vous pouvez les localiser sur la carte en cliquant sur le bouton "Localiser sur la carte".
  • Cela affichera les waypoints saisie sur la carte Google Maps.
  • Vous pourier aussi telecherger le PDF qui contient les services suivants :
    • Les laltitudes et longitudes saisie
    • Les routes maritines
    • Les distances maritines

Utiliser la géolocalisation :

  • Vous pouvez également utiliser la géolocalisation de votre navigateur pour ajouter votre position actuelle
  • Cliquez sur le bouton "Ma Position" pour Vous afficher votre position en laltitude et en longitude .
